# Proje Özeti — Zeynep'i Çoook Seviyorum ♡

## Temel Amaç
Murat'ın sevgilisi Zeynep için 1 aylık yıl dönümü hediyesi olarak hazırlanan kişisel, romantik bir web sitesi.

## Proje Adı
**"Zeynep'i Çoook Seviyorum ♡"**

## Kişiler
- **Murat** — Siteyi yapan (erkek taraf)
- **Zeynep** — Siteyi alacak (kız taraf)
- **Birlikte oldukları tarih:** 26 Mart 2026

## Temel Özellikler
1. **Hero bölümü** — Romantik görsel + gün sayacı ("X Gündür Birlikte")
2. **Spotify Çalma Listesi** — 3 öne çıkan şarkı + Spotify linki
3. **Murat Hakkında Bilmen Gereken 10 Şey** — İki sütunlu interaktif tablo (Murat'ın gerçekleri + Zeynep'in dolduracağı input alanları)
4. Footer

## Kapsam Dışı
- Kupon bölümü (kullanıcı talep etmedi)

## Dil
Tamamen Türkçe

## Teknoloji
- Saf HTML + Tailwind CSS (CDN)
- Vanilla JavaScript (gün sayacı için)
- Google Fonts (Noto Serif + Plus Jakarta Sans)
- Material Symbols
